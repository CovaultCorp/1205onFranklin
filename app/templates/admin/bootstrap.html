Bootstrap Users From UniFi

Run UniFi reconciliation first, then export all UniFi users, complete the local registry columns, and import the CSV. This workflow updates the local registry only.

Export All UniFi Users CSV Download Reference ZIP

Workflow

  1. Run UniFi reconciliation first.
  2. Export all UniFi users.
  3. Open the CSV.
  4. For each user to add or update, fill in promote or update_existing, Company, Suite, desired UniFi Access Policy, and desired UniFi User Group columns.
  5. Import the CSV.
  6. Rerun UniFi reconciliation.
  7. Review remaining conflicts.

UniFi Access does not use the app's internal Access Profile field. For bootstrap, use UniFi Access Policy and UniFi User Group columns. Access Profiles are optional local templates that may be used later.
