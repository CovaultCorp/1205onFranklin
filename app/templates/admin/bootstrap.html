Workflow

  1. Run UniFi reconciliation first.
  2. Export all UniFi users.
  3. Open the CSV.
  4. For each user to add or update, fill in promote or update_existing, company_id or company_name, local_suite_id or local_suite_number, desired UniFi Access Policy, and desired UniFi User Group columns.
  5. Upload a small test batch first.
  6. Review the preview screen for changed fields and validation errors.
  7. Commit the preview batch.
  8. Rerun UniFi reconciliation.
  9. Review remaining conflicts.

AccessProfile is a local optional template concept. UniFi Access uses Access Policies and User Groups. Bootstrap imports do not require AccessProfile.
